# 🪳 Tap Cockroach 全面改造提案
## 撳蟲大挑戰 2.0 - 完整設計規格書

---

## 一、項目概述

### 1.1 設計目標
- **目標年齡層**: 8-12歲兒童
- **核心玩法**: 點擊消滅害蟲，結合技能樹、寵物系統、成就系統的 RPG 風格休閒遊戲
- **故事模式**: 傳統中文敘事，引導玩家通過10個主題關卡

### 1.2 現有系統分析
| 現有系統 | 問題點 | 改造方向 |
|---------|--------|----------|
| 10個關卡 | 無主題特色，純數值遞增 | 主題化場景與視覺 |
| 4種蟑螂類型 | 類型太少互動性低 | 保留並擴展互動 |
| 生命/時間/分數 | 基礎三軸 | 加入技能減傷/計時加成 |
| 無寵物系統 | - | 新增3種寵物 |
| 無技能樹 | - | 新增6技能 |
| 無成就系統 | - | 新增10成就 |
| 無故事模式 | - | 關卡間敘事 |

## 二、故事模式設定

### 2.1 主線劇情（傳統中文）

**序章：廚房危機**
> 阿偉最愛的廚房突然被蟑螂大軍入侵了！這些邪惡的害蟲不僅偷吃餅乾碎片，還在角落裡開派對！快拿起你的拖鞋，成為滅蟑小勇士，拯救廚房的和平！

**關卡1-2：廚房大作戰**
- 場景：瓷磚地板、木製櫥櫃、食物殘渣
- 敵人：普通蟑螂、麵包蟲
- 結局：擊敗廚房首領「大黑」

**關卡3-4：浴室驚魂**
> 糟糕！蟑螂們發現了浴室的水龍頭！潮濕的環境讓它們繁殖得更快了...

- 場景：瓷磚牆面、水漬、肥皂泡泡
- 敵人：普通蟑螂、潮蟲（新增）、快速蟑螂
- 結局：擊敗浴室領主「濕濕」

**關卡5-6：臥室噩夢**
> 深夜，蟑螂們趁你睡覺時爬上了床！那些在黑暗中移動的黑影實在太可怕了...

- 場景：床單、枕頭、黑暗角落
- 敵人：普通蟑螂、 giant 蟑螂（視覺放大）、golden 能量球
- 結局：擊敗臥室霸主「黑影」

**關卡7-8：車庫保衛戰**
> 蟑螂大軍轉移到車庫！油污和機械零件是它們最愛的食物...

- 場景：水泥地面、機油漬、工具架
- 敵人：強化蟑螂、快速蟑螂、巨大蟑螂
- 結局：擊敗車庫首領「油膩」

**關卡9-10：害蟲總動員**
> 最終決戰！所有蟑螂集合起來組成超級害蟲軍團！這是拯救家園的最後一戰！

- 場景：所有之前關卡的混合元素
- 敵人：所有類型混合、精英怪
- 結局：戰勝「蟑螂皇后」，家園恢復和平！🏆

## 九、數值平衡建議

### 9.1 早期遊戲（關卡1-3）
- 較慢生成速度（1.0-1.5/s）
- 較多生命（4-5條）
- 較長時間（55-60秒）
- 目標：建立信心，教學式難度

### 9.2 中期遊戲（關卡4-6）
- 中等生成速度（1.7-2.3/s）
- 中等生命（3-4條）
- 中等時間（50秒）
- 目標：策略性資源管理

### 9.3 後期遊戲（關卡7-10）
- 高生成速度（2.6-3.5/s）
- 較少生命（2-3條）
- 較短時間（35-45秒）
- 目標：技巧與速度考驗
